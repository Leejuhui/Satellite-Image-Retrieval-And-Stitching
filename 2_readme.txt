Packages required to run the code:
*requests
*cv2
*argparse
*urllib
*numpy

How to run:
	
	Run this command in commandline: 
		python map_tile.py

	Or customize the input and output paths with flags: 
		Python 
			map_tile.py
			-- latitude1 12.34
			-- longitude1 45.78
			-- latitude2 56.78
			--longitude2 89.10
			--	release/debug
